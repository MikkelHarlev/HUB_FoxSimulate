JenXML - XML-Call: CustArtDelete()

The Operatordelete Call delete one item of the Operator list, or the whole list.

In dependence you want to delete you have to send a different option and you have to add members.

We suggest to use the OperatorLock before you use the OperatorDelete to avoid inconsistence.


Parameters:

Parameter

Data Type

Description
option <base64> Option to decide what should be cleared:
  • all (YWxs) -> Delete the whole List
  • Customer (Q3VzdG9tZXI=) -> Delete a customer with all articles
  • Article (QXJ0aWNsZQ==)-> Delete a single article
CuName <base64> Name of the customer to delete
( only with option Customer )
CuNumber <int> Number of the customer to delete
( only with option Customer or Article )
ArtName <base64> Name of the article to delete
( only with option Article )
ArtNumber <int> Number of the article to delete
(only with option Article )
