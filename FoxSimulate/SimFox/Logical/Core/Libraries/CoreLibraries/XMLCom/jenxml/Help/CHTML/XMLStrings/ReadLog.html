JenXML - XML-Call: ReadLog()

The ReadLog Call read Log entries from the Log File.


Parameters:

Parameter

Data Type

Description
LogName <base64> Name of the log which has to be read
defined via pLogCallName from JenLogInit()
option <base64> Option to decide where the function should start to read the entries
  • first (Zmlyc3Q=)  -> Start at the last position where SetLogPointer was called
  • next (bmV4dA==) -> Start with the next entry in the file

XML-Strings:
