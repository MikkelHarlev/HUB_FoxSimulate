DemoLib - Example of CalcChecksum in Automation Basic

Description of Example

This example of the CalcChecksum function calculates the checksum of the USINT array TestData and gives the result of the calculation to the UDINT variable Checksum1.

Example Code (cyclic)

(* cyclic program *)
SumStatus = CalcChecksum(DL_MODE_SUM, adr(TestData), sizeof(TestData), adr(Checksum1) )

Variable Declaration

Name Data Type Scope
SumStatus UINT local
TestData USINT[4] global
Checksum1 UDINT local
