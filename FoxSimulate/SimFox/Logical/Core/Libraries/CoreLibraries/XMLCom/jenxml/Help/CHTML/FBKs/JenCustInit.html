JenXML - JenCustInit()

JenCustInit() collect all data so that the PC can handle requests for the Customer Article list.
JenCustInit() has to be called in the Init part of a task. Otherwise you will get a cycletime violation.

The following actions are done:
    - connect to the address of the task interface

Parameters:

I/O

Parameter

Data Type

Description
IN Adr_Name_Cust_Struct UDINT Address of the name of the Interface of the CustArt-Lib
OUT status UINT Status Number (0 -> FUB finished succesfull)

Call Syntax (Automation Basic):

JenCustInit_0.Adr_Name_CustStruct= adr(Name_CustArtStruct)
JenCustInit_0 FUB JenCustInit()
