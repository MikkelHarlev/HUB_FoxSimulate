DemoLib - Example of CalcChecksum in Automation Basic

Description of Example

This example of the LogInit function shows how to initialise the LogLib.

Example Code (cyclic)

(* init program *)

LogInit_0.enable= TRUE;
LogInit_0.EntrysLogfile= 100;
LogInit_0.sizeLogbuffer= sizeof(Logbuffer);
LogInit_0.pNameDevice= adr('FileDevice');
LogInit_0.pNameFile= adr('FileName.csv');
LogInit_0.AdrLogbuffer= adr(Logbuffer);
LogInit_0.pNameWriteData= adr('LogData');

(* Call FUB *)
LogInit_0();

(* cyclic program *)
SumStatus = CalcChecksum(DL_MODE_SUM, adr(TestData), sizeof(TestData), adr(Checksum1) )

Variable Declaration

Name Data Type Scope
