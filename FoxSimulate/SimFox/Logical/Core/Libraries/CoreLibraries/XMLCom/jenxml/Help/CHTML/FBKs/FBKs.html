JenXML - Functions and Function Blocks

The JenXML-Library includes the following function blocks:

Call

Description
JenLogInit()
 Initializes the Library to read log Entrys
JenCustInit() Initializes the Library to handle requests for the CustArt-List  
JenOperInit() Initializes the Library to handle requests for the Operator-List 
JenRealTimeInit() Initializes the Library to handle real time requests
RegisterJenFunctions() Initialize all Funktions so that they can be called by a client
JenMessageInit() Stores the Log Buffer to the csv File 
JenCatTextInit() Set First Read Entry to actual position
JenXMLCyclic() Cyclic parts of Library
