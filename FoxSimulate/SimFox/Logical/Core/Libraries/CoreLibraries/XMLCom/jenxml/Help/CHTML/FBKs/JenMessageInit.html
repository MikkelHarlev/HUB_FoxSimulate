JenXML - JenMessageInit()

JenMessageInit() will initiate the ShowMessage function on the server.
JenMessageInti() has to be called in the Init part of a task. Otherwise you will get a cycletime violation.

The Message will be saved in the Variable which is given via Adr_Name_MsgString.
When a new Message is received, the Bit0 in the Statusvariable, given with Adr_Name_StatusVar will be cleared.
So the Statusvariable can be connect directly with the VC-Object.

The Statusvariable has to be set manually.

Parameters:

I/O

Parameter

Data Type

Description
IN Adr_Name_MsgString UDINT Address of the Name of the Variable where the received message will be saved
IN Adr_Name_StatusVar UDINT Address of the Name of the Variable which is connected to the status datapoint
OUT Status UINT Status Number (0 -> FUB finished succesfull)

Call Syntax (Automation Basic):

JenMessageInit_0.Adr_Name_MsgString = ADR( 'Message' )
JenMessageInit_0.Adr_Name_StatusVar = ADR( 'StatusVar' )
JenMessageInit_0 FUB JenMessageInit()
status = JenMessageInit_0.Status
