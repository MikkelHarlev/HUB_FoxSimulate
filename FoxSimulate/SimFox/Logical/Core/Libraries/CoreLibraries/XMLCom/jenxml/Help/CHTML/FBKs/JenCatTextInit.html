JenXML - JenCatTextInit()

JenCatTextInit() initialise the funktion to request the category texts
JenCatTextInit() has to be called in the Init part of a task. Otherwise you will get a cycletime violation.

Parameters:

I/O

Parameter

Data Type

Description
IN Adr_Name_RcpCommandStruct UDINT Address of the Name of the Recept command struct
OUT status UINT Status Number (0 -> FUB finished succesfull)

Call Syntax (Automation Basic):

JenCatTextInit_0.Adr_Name_RcpCommandStruct = ADR( 'RCfg' )
JenCatTextInit_0 FUB JenCatTextInit()
status = JenCatTextInit_0.status
