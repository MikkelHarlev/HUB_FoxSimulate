LogLib - Functions and Function Blocks

The LogLib-Library includes the following function blocks:

Call

Description
LogInit()
 Initializes the log lib
LogRead() Reads entries from Log Buffer  
LogWrite() Writes entries to Log Buffer  
LogStore() Stores the Log Buffer to the csv File 
LogSetLineRead() Set First Read Entry to actual position
