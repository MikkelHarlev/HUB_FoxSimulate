Jercp_lib - RcpPointInit()

The Function Block Searches the Pointer in the PLC and creates a Internal Info Structure. The Ident delivered back needs to be given to the RcpPointRef Function Block as Input Paramenter PointerIdent

Parameters

I/O

Parameter

Data Type

Description
IN pPointerName UDINT Address of a String containing the Recipe Pointer Name
OUT Status UINT Error Number
OUT PointerIdent UDINT Ident for RcpPointRef() Function Block
OUT ErrStringInfo STRING(32) Error Information in String Format

Call Syntax (Structured Text)

Status := RcpPointInit_0(pPointerName:=);
