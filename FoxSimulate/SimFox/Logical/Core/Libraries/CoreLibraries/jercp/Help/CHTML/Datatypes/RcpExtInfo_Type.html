RcpExtInfo_Type

Data Type which has to be used for the RcpInit() Function Block.

It provides Information where the Recipe Data could be accessed, how many recipes are currently in the memory, detailed Error Information if a Function Block returns with an Error.

Element

Data Type

Description
Ident UDINT Ident from RcpInit()
adrActRcpData UDINT Address where the actual recipes are located in the memory
NumberOfRecipes UINT Number of recipes in the memory
NumberOfFiles USINT Number of Recipe Files in the specified directory RcpDirList() - pDevice
MaxNumberOfFiles USINT Maximum allowed Number of Recipe Files - has to be set by the user!!
RcpStructName STRING(32) Name of the Recipe Structure given to RcpInit() Function Block
ErrStringInfo STRING(32) Error Information in Case a Function Block returns an Error
ErrStringDetail STRING(32) Detailed Error Information in Case a Function Block returns an Error
Status UINT Status in case a Function Block of the Library returns an Error
