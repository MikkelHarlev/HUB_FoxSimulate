Jercp_lib - ParScroInit()

This function block does the following things:

  • If "pFileNameParLanguage" given, then load the file into the memory (CSV Format)

  • Build up a List with all Parameter Names

  • Addtional if "pFileNameParLanguage" was loaded, then extend the List with these Entries. So it could be realized that the Parameters either could be shown with their names, or in a Clear Text - in different languages.

  • Initialize the ParScrollArray and the ParValueArray

The Function Block must be called in the Cyclic Part until the Status is unequal to 65535 - JERCP_FUB_BUSY!

Parameters

I/O

Parameter

Data Type

Description
IN Enable BOOL Enables the Function Block
IN Ident UDINT Ident from RcpInit()
IN pFileNameParLanguage UDINT Address of a String containing the Parameter Language Switching Filename (optional)
IN pParScrollArray UDINT Address of a String Array which is used to Scroll through all Recipe Parameters
IN pParValueArray UDINT Address of a String Array which is used to show the Value of the Recipe Parameters
IN pParNameArray UDINT Address of a String Array which is used to Scroll through all Recipe Names
IN pParNameString UDINT Address of a String which contains the Name of the Structure Element (STRING) which contains the Recipe Name
IN pActParValue UDINT Address of the String is used to show and edit the value of the actual selected parameter
IN pDevice UDINT Address of a String containing the Device Name -> FileIo
IN SizeParScrollArray UINT Size of the String Array used to scroll through the Parameters
IN SizeParValueArray UINT Size of the String Array used to show the Value of the Parameters
IN SizeParNameArray UINT Size of the String Array used to show the Recipe Names
IN SizeActParValue UINT Size of the String used to show and edit the actual selected paramter
IN LinesParScrollArray UINT Number of Elements in the String Array
IN LinesParNameArray UINT Number of Elements in the String Array
OUT Status UINT Error Number
OUT ErrStringInfo STRING(32) In case of Error Inforation is given in this String
OUT ErrStringDetail STRING(32) In case of Error Detailed Information is given in this String

Call Syntax (Structured Text)

ParScroInit_0(Enable:=, Ident:=, pFileNameParLanguage:=, pParScrollArray:=, pParValueArray:=, pParNameArray:=, pParNameString:=, pDevice:=, SizeParScrollArray:=, SizeParValueArray:=, SizeParNameArray:=, LinesParScrollArray:=, LinesParNameArray);
