JenXML - JenXMLCyclic()

JenXMLCyclic controlles the dataexchange between JenXML and the Taskinterfaces.
JenXMLCyclic has to be called cyclically.

Parameters:

I/O

Parameter

Data Type

Description
OUT status UINT Status Number (0 -> FUB finished succesfull)

Call Syntax (Automation Basic):
