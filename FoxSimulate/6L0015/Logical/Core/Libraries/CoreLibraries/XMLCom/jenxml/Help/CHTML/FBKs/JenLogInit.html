JenXML - JenLogInit()

JenLogInit() collect all data so that the PC can read out log data.
JenLogInit() has to be called in the Init part of a task. Otherwise you will get a cycletime violation.

The following actions are done:
    - Check if LogInit() was called successfully
    - Collect all needed data, like address of task interface

Important:
Before you call JenLogInit() make sure that LogInit() has be called successfully!

Parameters:

I/O

Parameter

Data Type

Description
IN pLogCallName UDINT Address of the name which is used for call the function from the server
IN pLogIdent_JenVar UDINT Ident given from LogInit() which has all data for JenXML
IN Adr_Name_ReadVar UDINT Address of the name of the Variable where the LogRead will save the data
IN Adr_Name_CommandStruct UDINT Address of the name of the task interface, the variable has to be a JenLogging_Type
OUT Status UINT Status Number (0 -> FUB finished succesfull)

Call Syntax (Automation Basic):

JenLogInit_0.pLogCallName= adr(LogCallName)
JenLogInit_0.pLogIdent_JenVar= LogInit_0.Ident_JenXML
JenLogInit_0.Adr_Name_ReadVar= adr(ReadVar)
JenLogInit_0.Adr_Name_CommandStruct= adr(CommandStruct)
JenLogInit_0 FUB JenLogInit()
