JenXML - JenOperInit()

JenOperInit() collect all data so that the PC can handle requests for the Customer Article list.
JenOperInit() has to be called in the Init part of a task. Otherwise you will get a cycletime violation.

The following actions are done:
    - connect to the address of the task interface

Parameters:

I/O

Parameter

Data Type

Description
IN Adr_Name_OperatorStruct UDINT Address of the name of the interface of the Operator-List
OUT Status UINT Status Number (0 -> FUB finished succesfull)

Call Syntax (Automation Basic):

JenOperInit_0.Adr_Name_OperatorStruct= adr(Name_OperatorStruct)
JenOperInit_0 FUB JenOperInit()
