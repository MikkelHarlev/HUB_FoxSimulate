JenXML - Status Numbers

There exists two types of Status Numbers, one has to be read from the function call on the PLC, the other will be send back as Fail-Response via XML.

XML Response:

50000 The Function is not initialized
50001 one or more needed Parameter is missing, remeber: functions are case sensitive 
50002 one or more Parameter has the wrong datatype
50003 Log functions: Log with specific name not found
50404 Log functions: No data is available
50405 Real time function: Structure with specific name not found
50406 Show message: There is already a message shown on the screen
65535 function is already in use, try again a bit later
other error from a function call, see help of specific function blocks
