JenXML - JenRealTimeInit()

JenRealTimeInit() collect all data so that the PC can handle requests for the Customer Article list.
JenRealTimeInit() has to be called in the Init part of a task. Otherwise you will get a cycletime violation.

The following actions are done:
    - create answer code for the specific variable

It is possible to initialize more than one Variable but they must not have the same name.

Parameters:

I/O

Parameter

Data Type

Description
IN pRealTimeCallName UDINT Pointer to the Name which the PC will use
IN pName_RealTimeStruct UDINT Pointer to the Name of the Variable/Array/Struct which should be used as datasource
OUT status UINT Status Number (0 -> FUB finished succesfull)

Call Syntax (Automation Basic):

JenRealTimeInit_0.pRealTimeCallName= ADR('CallName')
JenRealTimeInit_0.pName_RealTimeStruct= ADR('VariableName')
JenRealTimeInit_0 FUB JenRealTimeInit()
