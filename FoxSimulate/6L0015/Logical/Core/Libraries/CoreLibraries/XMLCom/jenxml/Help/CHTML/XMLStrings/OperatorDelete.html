JenXML - XML-Call: OperatorDelete()

The Operatordelete Call delete one item of the Operator list, or the whole list.

If a Single operator should be deleted, then you have to send another option with the Operator Name and the Operator Number.

We suggest to use the OperatorLock before you use the OperatorDelete to avoid inconsistence.


Parameters:

Parameter

Data Type

Description
option <base64> Option to decide what should be cleared:
  • all (YWxs) -> Delete the whole List
  • Operator (T3BlcmF0b3I=) -> Delete a single operator
OpName <base64> Name of the operator to delete
( only with option Operator )
OpNumber <int> Number of the operator to delete
( only with option Customer or Article )


XML-Strings:

<?xml version="1.0" ?>
<methodCall>
    <methodName>OperatorDelete</methodName>
    <params>
        <param>
