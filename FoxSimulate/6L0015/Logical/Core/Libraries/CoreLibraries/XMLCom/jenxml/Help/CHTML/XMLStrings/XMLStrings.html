JenXML - XML-Strings

The JenXML-Library can handle the following method Calls:

Important:
The strings will be handled case sensitive!
All variables which are received but not needed will be ignored.
If a needed variable is missing you will get an error 50001.
If a needed variable has a false type, you will get an error 50002.


Call

Description
OperatorLoad Load the Operator List from the CSV File
OperatorSave Save the Operator List to the CSV File
OperatorLock Lock the Operator List for the user to prevent inconsistency
OperatorUnlock Unlock the Operator List for the user
OperatorDelete Delete either a single operator or the complete list
OperatorAdd Add an Operator in the operator list
CustArtLoad Load the customer&article list from the CSV File
CustArtSave Save the customer&article list to the CSV File
CustArtLock Lock the customer&article list to prevent inconsistency
CustArtUnlock Unlock the customer&article list
CustArtDelete Delete either a single customer, a single article or the complete list
CustArtAdd Add a customer or an article in the list
ReadLog Read log entries from the Log (NOT from the PLC-Log)
SetLogPointer Set that Read log with argument first will start with after the last read
GetRealTimeData Get the actual value of an variable/array/structure
GetTime Get the actual time of the PLC
SetTime Set the actual time of the PLC
GetCatTexts Get the category texts
ShowMessage Show a message on the display
