Jercp_lib - RcpGetMemInfo()

The RcpGetMemInfo Function retrieves the amount of bytes available in the Memory partition, and also the block which holds the biggest amount of free bytes.

 

I/O

Parameter

Data Type

Description
IN Ident UDINT Ident from RcpInit FUB
IN cmdGetMemInfo USINT trigger execution of function
OUT Status UINT Error Number
OUT NumberBytesFree UDINT Total amount of free bytes in the memory partition
OUT MaxBlockSizeFree UDINT Biggest available block in the memory partition
