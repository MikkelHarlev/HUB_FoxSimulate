jercp_lib Function Blocks and Functions

The jercp_lib contains the following function blocks and functions

Call Description
RcpInit() Initialize the recipe data. Create the CSV header with the given structure, create a database with all addresses and datatypes of all structure elements.
ParScroInit() Initialize the parameter scroll. Create a list with all parameternames for scrolling
RcpDirList() Build a list with the filenames of a directory
ParScroCyclic() Scroll in the list of parameternames, view their value and edit parameters; Switch between languages if configured
RcpFileListScroll() Scroll in the list of files built by RcpDirList()
RcpLoad() Load a Csv file. All recipes will be copied to the memory.
RcpSave() Save a Csv file. All recipes will be stored in a file.
RcpAdd() Add a recipe. A blank recipe will be added to the existing ones in the memory (only when working with dynamic allocated memory)
RcpCopy() Copy recipes from source index X a number or recipes Y to destination index Z
RcpNameList() Create a recipe name List to scroll within for the display (Recipe name is part of the Recipe)
RcpGetNames() Copy all recipe names into a string array
RcpPointInit() Initialize FBK when working with recipe pointers (call for each pointer!!)
RcpPointRef() References the recipe pointer (RcpPointInit must be done before!)
RcpDeInit() Release all allocated memory. Function for the _EXIT in C language
RcpGetMemInfo() Get Information from the Memory Partition
