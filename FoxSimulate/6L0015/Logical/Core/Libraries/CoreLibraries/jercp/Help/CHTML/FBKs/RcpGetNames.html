Jercp_lib - RcpGetNames()

This Function Block is doing the following things:

  • Copy all Recipe Names to a String Array.

Note: The string array should hold as many strings are recipes exist!

Parameters

I/O

Parameter

Data Type

Description
IN Enable BOOL Enables the Function Block
IN Ident UDINT Ident from RcpInit()
IN pRcpNameArray UDINT Address of a string array where the names will be written to
IN SizeRcpNameArray UINT Size of the string array
IN LinesRcpNameArray UINT Number of lines in the string array
OUT ErrStringInfo STRING(32) In case of Error Information is given in this String
OUT Status UINT Error Number

Call Syntax (Structured Text)

    RcpGetNames_0(Enable:=, Ident:=, pRcpNameArray:=, SizeRcpNameArray:=, LinesRcpNameArray:=);
