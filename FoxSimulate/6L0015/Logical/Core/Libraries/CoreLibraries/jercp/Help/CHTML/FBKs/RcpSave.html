Jercp_lib - RcpSave()

This Function block does the following things:

  • Calculate and Allocate Memory which is needed to store all recipes in CSV Data
  • Convert Recipe by Recipe from Structured Data into CSV Format
  • Create the File given in pFileName
  • Copy the Header from RcpInit() to the File
  • Copy the CSV Data to the File
  • Release the allocated Memory
  • Close the File

The Function Block must be called in the Cyclic Part until the Status is unequal to 65535 - JERCP_FUB_BUSY!

Parameters

I/O

Parameter

Data Type

Description
IN Enable BOOL Enables the function
IN Ident UDINT Ident from RcpInit()
IN pDevice UDINT Address of a String containing the Device Name -> FileIo for detailed Info
IN pFileName UDINT Address of a String containing the File Name
IN optHeaderOnly USINT 0... Save Header and Recipe Data
1... Save only Header
OUT Status UINT Error Number
OUT ErrStringInfo STRING(32) In case of Error Information is given in this String
OUT ErrStringDetail STRING(32) In case of Error detailed Information is given in this String

Call Syntax (Structured Text)

RcpSave_0(Enable:=, Ident:=, pDevice:=, pFileName:=);
