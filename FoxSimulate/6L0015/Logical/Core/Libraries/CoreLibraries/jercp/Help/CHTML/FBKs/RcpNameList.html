Jercp_lib - RcpNameList()

This Function Block is doing the following things:

  • Scrolling in the Recipe Name List List, but only possible if configured in ParScroInit() FBK

Parameters

I/O

Parameter

Data Type

Description
IN Enable BOOL Enables the Function Block
IN Ident UDINT Ident from RcpInit()
IN cmdScrollUp USINT Scroll Up in the Parameter List
IN cmdScrollDown USINT Scroll Down in the Parameter List
OUT SelectedItem UINT Output for Visual Components List Box
OUT RcpNumber UINT Index of the Recipe (starting from 1!!!)
OUT ErrStringInfo STRING(32) In case of Error Information is given in this String
OUT Status UINT Error Number

Call Syntax (Structured Text)

    RcpNameList_0(Enable:=, Ident:=, cmdScrollUp:=, cmdScrollDown:=);
