Jercp_lib - ParScroCyclic()

This Function Block is doing the following things:

  • Scrolling in the Parameter List

  • Copy the Parameter Names to the ParScrollArray

  • Copy the Parameter Values to the parValueArray

  • Change the Language if configured

  • Change the Value of a Parameter

Parameters

I/O

Parameter

Data Type

Description
IN Enable BOOL Enables the Function Block
IN Ident UDINT Ident from RcpInit()
IN cmdUp USINT Scroll Up in the Parameter List
IN cmdDown USINT Scroll Down in the Parameter List
IN cmdLangSwitch USINT Switch to other Language "ActLanguage" if configured in ParScroInit()
IN cmdEditPar USINT Edit a Parameter - "ActParValue" will be written directly to the Recipe Data in the Memory ot the Recipe Number given by SelectRcpNumber.
IN cmdRcpChange USINT After the Recipe Number is select with Parameter SelectRcpNumber, this command has to be set.
IN ActLanguage USINT Actual Language
IN SelectRcpNumber UINT Recipe Number which will be edited with cmdEditPar
IN ShowLevel USINT Actual Password Level: Only parameter configured with PWL equal or lower will be shown
OUT Status UINT Error Number
OUT ActParNumber UINT Number of the Actual Parameter
OUT SelectedItem UINT Output for Visual Components List Box
OUT ErrStringInfo STRING(32) In case of Error Information is given in this String

Call Syntax (Structured Text)

    ParScroCyclic_0(Enable:=, Ident:=, cmdUp:=, cmdDown:=, cmdLangSwitch:=, cmdEditPar:=, cmdRcpChange:=, ActLanguage:=, SelectRcpNumber:=);
