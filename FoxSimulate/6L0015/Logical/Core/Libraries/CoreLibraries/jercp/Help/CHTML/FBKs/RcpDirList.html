Jercp_lib - RcpDirList()

  • Get Information about the Directory given in pDevice

  • Allocate memory according the number of Files in this directory

  • Read out Filename per Filename

  • If optional Filter for Extension given in pFileExtFilter, then each File will be checked for this extension. Only Files with this extension are copied into the allocated File List.

The Function Block must be called in the Cyclic Part until the Status is unequal to 65535 - JERCP_FUB_BUSY!

 

Parameters

I/O

Parameter

Data Type

Description
IN Enable BOOL Enables the Function Block
IN Ident UDINT Ident from RcpInit() Function Block
IN pDevice UDINT Address of a String containing the Device Name -> FileIo
IN pFileExtFilter UDINT Address of a String which contains the extension of Files which should be filtered. (Optional)
OUT Status UINT Error Number
OUT ErrStringInfo STRING(32) In case of Error Information is given in this String

Call Syntax (Structured Text)

    RcpDirList_0(Enable:=, Ident:=, pDevice:=, pFileExtFilter:=);
