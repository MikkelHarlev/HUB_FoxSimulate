Jercp_lib - RcpFileListScroll()

  • Get Filenames from the Allocated Memory by RcpDirList() and copy then to the FileNameArray[].

  • Scroll in this List of Filenames

Parameters

I/O

Parameter

Data Type

Description
IN Enable BOOL Enables the Function Block
IN Ident UDINT Ident from RcpInit() Function Block
IN cmdUp USINT Command Scroll up in the File List
IN cmdDown USINT Command Scroll down in the File List
IN cmdInit USINT Command Initialize File List
OUT Status UINT Error Number
OUT SelectedItem USINT Output for Visual Components List Box
OUT ActFileNumber UINT Actual Filenumber in the List
OUT ActFileName STRING(32) Actual Filename where the cursor of the Listbox is placed on
OUT ErrStringInfo STRING(32) In case of Error Information is given in this String

Call Syntax (Structured Text)

    RcpFileListScroll_0(Enable:=, Ident:=, cmdUp:=, cmdDown:=, cmdInit:=);
