Jercp_lib - RcpPointRef()

The function references the given Pointer (PointerIdent from RcpPointInit FBK) to the selected Rcp Number.

Parameters

I/O

Parameter

Data Type

Description
IN RcpInitIdent UDINT Ident from RcpInit() Function Block
IN PointerIdent UDINT Number of the Recipe to which should be referenced
IN Index UINT Index of the Recipe which should be accessed
OUT Status UINT Error Number

Call Syntax (Structured Text)

Status := RcpPointRef_0(RcpInitIdent:=, PointerIdent:=, Index:=);
