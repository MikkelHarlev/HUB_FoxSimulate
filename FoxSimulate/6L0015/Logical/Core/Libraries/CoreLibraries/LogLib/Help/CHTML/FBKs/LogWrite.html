LogLib - LogWrite()

LogWrite() writes a new entry to the log buffer.

Each time the FUB is called with Enable = 1, a log entry is added (through the WriteDatatype given to LogInit()), to the FIFO buffer.
Every Task can have his own write structure, but all read and write structures must have the same type!

This FBK has to be called until Status != LOGLIB_BUSY

Parameters:

I/O

Parameter

Data Type

Description
IN enable BOOL Enables function block (FBK)
IN ident UDINT Ident from LogInit()
IN AdrWriteData UDINT Address of the write structure
OUT status UINT Status Number (0 -> FUB finished succesfull)

Call Syntax (Automation Basic):

LogWrite(enable, ident, AdrWriteData, status )
 

Examples:

Automation Basic
